# 推送到远程仓库

- git init
- git add . //存放暂存区
- git commit -m "描述" //提交到本地仓库
- git remote add origin https://github.com/740255321/git01.git 绑定远程仓库
- git push -u origin master //推送到远程仓库

- git pull //(拉取远程仓库更新代码)



git ls-files //查看本地仓库文件
