# 外部样式表

TIP

当样式需要应用于很多页面时，外部样式表将是理想的选择,

<head>
  <link rel="stylesheet" type="text/css" href="mystyle.css" />
</head>

# 内部样式表

TIP

当单个文档需要特殊的样式时，就应该使用内部样式表

<head>
  <style>
    hr {
      color: sienna;
    }
    p {
      margin-left: 20px;
    }
    body {
      background-image: url("images/back40.gif");
    }
  </style>
</head>

# 内联样式

TIP

请慎用这种方法,仅当样式仅需要在一个元素上应用一次时

<p style="color:sienna;margin-left:20px">这是一个段落。</p>

TIP

配合之前的标签、id、class 综合运用

TIP

使用场景:多个标签样式相同，一个选择器选不玩

th,
ul {
  margin: 0;
  padding: 0;
}

TIP

当两个规则都作用到了同一个 html 元素上时，如果定义的属性有冲突，那么应该用谁的值的，CSS 有一套优先级的定义。

!important > 行内样式 > id > 后代 > 类 > 标签名 > * > 标签默认样式 > 继承的样式

# CSS 盒子模型

TIP

所有 HTML 元素可以看作盒子,CSS 盒模型本质上是一个盒子 封装周围的 HTML 元素，它包括：边距，边框，填充，和实际内容。 盒模型允许我们在其它元素和周围元素边框之间的空间放置元素。

盒子宽度 = width + padding 左右 + border 左右

盒子高度 = height + padding 上下 + border 上下

# CSS 浮动

TIP

CSS 浮动会使元素向左或向右横向排列 浮动会脱离标准普通流的控制

<style>
  img {
    float: left;
  }
</style>
<div>
  <img />
  <img />
  <img />
  <img />
</div>

WARNING

注意:浮动之后，盒子因为脱离了标准文档流，撑不起父盒子的高度，导致父盒子高度塌陷。如果网页中出现了这种问题，会导致我们整个网页的布局紊乱。所以我们一定要清除浮动

# 清除浮动

方法1(推荐)
<div class="father clearfix">
  <div class="child1">盒子A</div>
  <div class="child2">盒子B</div>
</div>

.clearfix::after {
  content: "";
  display: block;
  clear: both;
}

方法2
<div class="father clearfix">
  <div class="child1">盒子A</div>
  <div class="child2">盒子B</div>
</div>

.clearfix {
  overflow: hidden;
}

# CSS 居中

# 行内元素居中

水平居中

<div class="container">hello world</div>

.container {
  text-align: center;
}

水平垂直居中

<div class="son">这是要居中的文字</div>

.son {
  height: 100px;
  text-align: center;
  line-height: 100px;
}

# 块级元素居中

水平居中

<div class="container">
  <div class="box"></div>
</div>

.box {
  width: 200px;
  height: 100px;
  background: red;
  margin: 0 auto;
}

垂直居中

<div class="container">
  <div class="box"></div>
</div>

.container {
  width: 200px;
  height: 100px;
  background: #000;
  display: flex;
  align-items: center;
}
.box {
  width: 50px;
  height: 50px;
  background: red;
}

# CSS 定位

为什么要使用定位？

1.某个元素可以自由的在一个盒子内移动位置，并且压住其他盒子。
2.当我们滚动窗口时，有一些盒子固定在屏幕的某个位置。
3.定位可以让盒子自由的在某个盒子内移动位置或者固定在屏幕中的某个位置，并且可以压住其他盒子

# 相对定位

position: relative;(相对定位)相对定位的元素，
通过top、bottom、left、right的设置相对于本身位置进行定位，
可以通过z-index进行层次分级；

# 绝对定位

absolute(绝对定位)：生成绝对定位的元素，相对于值不为static的第一个父元素进行定位，
且它是脱离正常文档流、不占位的；

# 固定定位

fixed(固定定位)：生成绝对定位的元素，相对于浏览器窗口进行定位，
元素的位置通过top、bottom、left、right属性进行规定。
