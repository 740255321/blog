# HTML5 新特性

# HTML5 新元素

标签 描述
<canvas> 标签定义图形，比如图表和其他图像。该标签基于 JavaScript 的绘图 API
<audio> 定义音频内容
<video> 定义视频（video 或者 movie）
<source> 定义多媒体资源 <video> 和 <audio>
<embed> 定义嵌入的内容，比如插件
<track> 为诸如 <video> 和 <audio> 元素之类的媒介规定外部文本轨道

# 新表单元素

标签 描述
<datalist> 定义选项列表。请与 input 元素配合使用该元素，来定义 input 可能的值。
<keygen> 规定用于表单的密钥对生成器字段。
<output> 定义不同类型的输出，比如脚本的输出。

# 新的语义和结构元素

标签 描述
<article> 定义页面独立的内容区域。
<aside> 定义页面的侧边栏内容。
<footer> 定义 section 或 document 的页脚。
<header> 定义了文档的头部区域。
<nav> 定义导航链接的部分。
<time> 定义导航链接的部分。

# 本地存储

# localStorage

  • localStorage - 用于长久保存整个网站的数据，保存的数据没有过期时间，直到手动去除。
  • localStorage 对象存储的数据没有时间限制。第二天、第二周或下一年之后，数据依然可用。

# 存数据

//存储名字为name值为caibin的变量
localStorage.setItem("name", "caibin");
// 等价于上面的命令
localStorage.name = "caibin";
// Storage {name: "caibin", length: 1}
localStorage;

# 读取数据

//caibin,读取保存在localStorage对象里名为name的变量的值
localStorage.getItem("name");
// "caibin"
localStorage.name;
//读取存储在localStorage上的所有数据
localStorage.valueOf();
// 读取第一条数据的变量名(键值)
localStorage.key(0);
//遍历并输出localStorage里存储的名字和值
for (var i = 0; i < localStorage.length; i++) {
  console.log(
    "localStorage里存储的第" +
      i +
      "条数据的名字为：" +
      localStorage.key(i) +
      ",值为：" +
      localStorage.getItem(localStorage.key(i))
  );
}

# 清空 localStorage

// undefined
localStorage.clear();
// Storage {length: 0}
localStorage;

# 删除某个变量

//undefined
localStorage.removeItem("name");
// Storage {length: 0} 可以看到之前保存的name变量已经从localStorage里删除了
localStorage;

# 检查 localStorage 里是否保存某个变量

// 这些数据都是测试的，是在我当下环境里的，只是demo哦～
localStorage.hasOwnProperty("name"); // true
localStorage.hasOwnProperty("sex"); // false

# 将 JSON 存储到 localStorage 里

var students = {
  xiaomin: {
    name: "xiaoming",
    grade: 1,
  },
  teemo: {
    name: "teemo",
    grade: 3,
  },
};
//将JSON转为字符串存到变量里
students = JSON.stringify(students);

//将变量存到localStorage里
localStorage.setItem("students", students);

var newStudents = localStorage.getItem("students");
newStudents = JSON.parse(students); //转为JSON
console.log(newStudents); // 打印出原先对象

# sessionStorage

  • sessionStorage 用法于 localStorage 类似
  • sessionStorage - 用于临时保存同一窗口(或标签页)的数据，在关闭窗口或标签页之后将会删除这些数据。

# 存数据

// 用途：将value存储到key字段
sessionStorage.setItem("key", "value");

# 取数据

// 用途：获取指定key本地存储的值
sessionStorage.getItem("key");
