· Milk

2. Milk

# HTML 元素

TIP

HTML 元素可分为：块元素、行内元素、行内块元素

# 块元素

特点:独占一行,可设置宽高

<div>hello</div>
<p>hello</p>
<h1>hello</h1>

# 行内元素

特点:不独占一行,不可设置宽高

<span>hello</span>
<a>hello</a>
<b>hello</b>

# 行内块元素

特点:不独占一行,可设置宽高

<img />
<textarea></textarea>
<input />

WARNING

元素之间的转换
display:inline（转为行内元素）
display:inline-block（转为行内块元素）
display:block（转为块元素）
display:none（隐藏 不显示）
注意：当元素浮动（float）时会转化成行内块元素特点。

# HTML 表格

# 基本表格结构

<table border="1">
  <tr>
    <td>row 1, cell 1</td>
    <td>row 1, cell 2</td>
  </tr>
  <tr>
    <td>row 2, cell 1</td>
    <td>row 2, cell 2</td>
  </tr>
</table>

# HTML 表格标签

标签 描述
<table> 定义表格
<th> 定义表格的表头
<tr> 定义表格的行
<td> 定义表格单元
<caption> 定义表格标题
<colgroup> 定义表格列的组
<thead> 定义表格的主体
<tbody> 定义表格的行
<tfoot> 定义表格的页脚

# HTML 表单标签

标签 描述
<form> 定义供用户输入的表单
<input> 定义输入域
<textarea> 定义文本域 (一个多行的输入控件)
<label> 定义了 <input> 元素的标签，一般为输入标题
<fieldset> 定义了一组相关的表单元素，并使用外框包含起来
<legend> 定义了 <fieldset> 元素的标题
<select> 定义了下拉选项列表
<optgroup> 定义选项组
<option> 定义下拉列表中的选项
<button> 定义一个点击按钮
<datalist> 指定一个预先定义的输入控件选项列表
<keygen> 定义了表单的密钥对生成器字段
<output> 定义一个计算结果
